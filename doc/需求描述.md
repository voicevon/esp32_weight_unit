# 需求描述 (Requirement Description)

## 1. 项目概述
本项目旨在开发一个基于 ESP32 的智能称重控制单元。该单元集成了高精度称重传感器、OLED 显示屏、舵机控制以及 RS485 通信接口，用于实现自动化称重与分拣控制。

## 2. 核心功能需求

### 2.1 称重功能 (Weighing)
- **传感器**: 使用 HX711 模块读取称重传感器数据。
- **功能**:
    - 实时读取重量值。
    - 支持去皮 (Tare) 操作。
    - 支持通过参数调整比例系数 (Calibration)。

### 2.2 显示功能 (Display)
- **硬件**: 0.91寸 OLED 显示屏 (SSD1306, 128x32, I2C接口)。
- **内容**:
    - 实时显示当前重量（单位：g）。
    - 显示系统状态（如初始化中、就绪）。

### 2.3 执行机构 (Actuator)
- **硬件**: 舵机 (Servo)。
- **功能**:
    - **OPEN**: 控制舵机转动到指定开启角度（默认90度）。
    - **CLOSE**: 控制舵机转动到指定关闭角度（默认0度）。

### 2.4 通信功能 (Communication)
- **接口**: RS485 (Modbus 物理层，自定义 ASCII 协议)。
- **指令集**:
    - 接收 `OPEN` -> 执行舵机开启，回复 `OK:OPEN`
    - 接收 `CLOSE` -> 执行舵机关闭，回复 `OK:CLOSE`
    - 接收 `TARE` -> 执行去皮，回复 `OK:TARED`
    - 接收 `GET_WEIGHT` -> 回复当前重量 `W:123.45`

## 3. 硬件清单
1.  **主控**: ESP32 Dev Module
2.  **称重模块**: HX711 + 压力传感器
3.  **显示屏**: SSD1306 OLED (I2C)
4.  **通信模块**: RS485 转 TTL 模块 (MAX485)
5.  **舵机**: SG90 或类似型号
