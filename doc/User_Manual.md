# 称重从站模块 (Weight Unit) 操作手册

本文档描述了 ESP32 称重模块单机版的界面交互机理与按键操作说明。

## 1. 界面与显示 (UI)
屏幕顶部状态栏（Header）固定显示基础信息：
- **ID: xx**：当前设定好的 Modbus 从站站号（Slave ID）。
- **星号 (*)**：右上角的通信脉冲指示灯。当接收到主站 Modbus 控制指令时，该星号会亮起，用于直观判断通信是否正常。

主屏幕会根据设备当前的工作状态（SystemState）切换显示：
- **正常运行界面**：中央大字体显示实时重量（单位：g）。
- **站号设置界面**：显示 `CHANGE ID >` 以及当前调整中的数字，下方提示 `[PRESS ADD / HOLD SAVE]`。
- **标定界面**：短暂显示 `CALIBRATING...`，表示正在提取传感器数据进行内部重量特征校准。
- **通信诊断**：显示 RS485 测试模式下的收发数据。

## 2. 实体按键操作指南

设备带有一个物理操控按键，基于时间的判断分为 **短按**（快速按下并松开）和 **长按**（持续按下 > 2秒）两种动作。

### A. 正常态 (Normal Run)
- **短按 (< 2秒)**：**去皮 (Tare) / 清零**。丢弃当前托盘皮重，将当前显示的重量归 0。
- **长按 (> 2秒)**：**进入站号配置**。系统切换至设置模式（Config ID），您可以给设备分配一个新的 Modbus 地址。

### B. 站号配置态 (Config ID)
- **短按**：**增加站号 (Add)**。按下一次，站号数值 +1。站号可调节的合法范围为 1 ~ 32，加到 32 后再按将循环回到 1。
- **长按 (> 2秒)**：**保存并退出 (Save)**。系统会将屏幕上的站号存入掉电不丢失的 EEPROM (Preferences) 中，基于新站号重新初始化 RS485 Modbus 端口，随后自动回到正常运行界面。

## 3. 标定 (Calibration) 与 Modbus 控制协议

由于本模块主要作为分布式称重网络的一环，为兼顾多模块的协同效率，**重量标定过程主要依赖 RS485 通过主站远程触发**，而非按键触发。

**Modbus 远程下发指令清单 (控制寄存器)：**
- `指令 1`：**Open** - 驱动舵机转到 90° (打开落料门)
- `指令 2`：**Close** - 驱动舵机复位至 0° (关闭落料门)
- `指令 3`：**Tare** - 远程去皮/清零
- `指令 4`：**Calibrate** - **触发标定**。主站需要确保设备上刚好放置了标准的基准砝码（系统中预设 `_calWeights` 索引目前采用 500g 档位），发送此指令后，分站自动进入校准周期，刷新放大器倍率。

> **功能拓展提示**：
> UI 库（`DisplayComponent`）中其实已经预留了 `STATE_MENU_SELECT_WEIGHT` 这套可视化的标定重量选择界面（比如 `CALIBRATE WEIGHT -> 500g`）。如果您日后确实需要“完全不依赖主站，纯靠按键进行选重和标定”的脱机标定功能，我们将通过扩展现有的 `handleButton()` 逻辑（比如加入双击进入标定模式），把这段沉睡的 UI 逻辑串联起来。目前的设定是优先服务于主站集中管理。
